# LGBTQ Center Automation - User Guide

## Quick Start

1. **Install the Application**
   - Download `LGBTQ-Center-Automation-Setup.exe`
   - Double-click to run the installer
   - Follow the wizard prompts
   - Launch from desktop shortcut

2. **Initial Setup**
   - Open the application
   - Click "Connect Gmail" to link your email
   - Enter your center information
   - Customize email templates (optional)

3. **Add Your First Client**
   - Click "New Client"
   - Fill in client information
   - Set intake date and appointment date
   - Save - automation begins automatically!

## Features

- **Automatic Emails**: 4-email sequence sent automatically
- **Client Database**: Track all client interactions
- **Staff Dashboard**: See today's tasks at a glance
- **Email Templates**: Customize with visual editor
- **Secure & Private**: Your data stays local
