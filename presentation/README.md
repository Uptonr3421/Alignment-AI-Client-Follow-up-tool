# Presentation / Gift Page

This folder contains the landing page for the Cleveland LGBTQ Center.

## Files

- `index.html` - Main gift page (single file, self-contained)
- `assets/` - Images and logos

## Deployment

### Option 1: Vercel
```bash
npm i -g vercel
vercel --prod
```

### Option 2: Netlify Drop
1. Go to https://app.netlify.com/drop
2. Drag this folder onto the page
3. Done

### Option 3: GitHub Pages
1. Enable Pages in repo settings
2. Set source to this folder
3. Access at `https://yourusername.github.io/lgbtq-center-automation/`

## Customization

Edit `index.html` to update:
- Download links (point to GitHub Releases)
- Contact information
- Logo images
- Color scheme (CSS variables at top)

## Assets Needed

Place these in `assets/` folder:
- `alignment-ai-logo.png` - Your company logo
- `lgbt-center-logo.png` - Cleveland LGBTQ Center logo
- `nglcc-logo.png` - NGLCC certification badge
- `woman-at-desk.png` - Hero image (optional)
